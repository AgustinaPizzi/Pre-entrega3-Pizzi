¿COMO ARMO TU PIUMA? 👇

  • 1° Molde

    Los moldes son sumamente necesarios para comenzar a desarrollar una cartera un bolsos,una bandolera.Cada pieza cumple una función y es importante que estén correctos, así a la hora de comenzar con el armado y la confección no tenemos inconvenientes. Para realizar un molde necesitamos una Escuadra/Regla, Lápiz a mina y cartón. Utiliza unlápiz para poder corregir errores más fácilmente.

  • 2° Marcar el cuero

    Para realizar un corte perfecto es necesario marcar el cuero de manera correcta, es por eso que aconsejamos realizar los moldes en cartón, para que a la hora de dibujarlos sobre el cuero el material rígido nos permite no pasarnos ni corrernos del molde.

  • 3° Cortar el cuero

    El corte y el aparado de una cartera es fundamental en el producto terminado. Si las piezas que componen la cartera o el bolso están cortadas correctamente y sumamente prolijas, facilitarán la confección.

  • 4° Unir las piezas

    El pegamento que se utiliza es el cemento de contacto o de aparar, siempre, para unir pieza con pieza es necesario esparcir cemento en ambas piezas, dejar secar y luego unir. Para lograr ser prolijos en nuestros productos, utilizamos distintas maneras de cementar: Cementado simple y superpuesto

  • 5° Coser

    Para coser el cuero se utilizan distintas técnicas de costura, la más utilizada para costuras internas es el punto atrás o punto guarnicionero, es el que imita el punto de una máquina de coser. Luego tenemos distintas técnicas de costura que se aplican para hacer bolsos y carteras de cuero: Punto simple, punto cruzado doble aguja y punto guarnicionero doble aguja como las principales
